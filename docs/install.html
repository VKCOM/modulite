Установка Modulite в PHPStorm

JetBrains больше не может предоставлять услуги VK как компании из РФ, поэтому все наши плагины распространяются не через официальный Marketplace, а через наш собственный.

Settings → Plugins → ⚙️ → Manage plugin repositories:

Добавляем адрес https://idea-plugins.kphp.dev

Закрываем окно настроек через ОК и открываем настройки заново.

Теперь он должен находиться по запросу (иногда без иконки):

Плагин совместим с версиями 2022.2 и выше.

Рекомендуется иметь также KPHPStorm, чтобы когда классы встречаются внутри tuple(...) и других ненативных типах, Modulite их тоже обнаруживал и добавлял в require.

Если в проекте уже есть .modulite.yaml файлики, лучше вызвать переиндексацию.
